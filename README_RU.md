# Перевод промптов для Forge NEO / Stable Diffusion WebUI

Переводите **положительный промпт** в Forge NEO / A1111 WebUI одним нажатием клавиши или автоматически при генерации.

<img width="923" height="308" alt="47567" src="https://github.com/user-attachments/assets/342c8d23-5514-4c78-9f2b-8dfdc2c438c6" />

## Возможности

- Работает в `txt2img` и `img2img`
- Компактная строка переводчика рядом с полями промпта
- Язык `Откуда` поддерживает `Автоопределение`
- Язык `Куда` по умолчанию — `English`
- `Alt+Q` — мгновенный перевод текущего положительного промпта
- `Alt+W` — обмен языков `Откуда` и `Куда`
- Режим **Автоперевод**: при нажатии Generate временно подставляет переведённый промпт, запускает генерацию, затем восстанавливает исходный текст
- Перевод выполняется только для **положительного промпта** (негативный не затрагивается)
- Безопасная обработка ошибок перевода (генерация не прервётся)
- Кэш переводов в памяти на бэкенде

## Установка

1. Скопируйте папку в директорию расширений WebUI:
   - `stable-diffusion-webui/extensions/prompt-translator`
2. Перезапустите Forge NEO / WebUI.
3. Зависимости установятся через `install.py` (`deep-translator`, `langdetect`).

Альтернативный способ (рекомендуется):
1. Откройте WebUI -> вкладка `Extensions` -> `Install from URL`.
2. Вставьте ссылку на репозиторий:
   - `https://github.com/abzaloff/small-translate.git`
3. Нажмите `Install`, затем перезапустите Forge NEO / WebUI.

## Использование

Состояние по умолчанию при загрузке Forge UI:
- `Автоперевод`: ВЫКЛ
- `Откуда`: Автоопределение
- `Куда`: English

Типичный сценарий:
1. Напишите промпт на своём языке.
2. Нажмите `Alt+Q` для однократного перевода.
3. Или включите **Автоперевод** — промпт будет переводиться только на время генерации, затем восстанавливаться.
4. Используйте `Alt+W` или кнопку обмена для смены языков местами.

## Об автоопределении языка

В режиме Автоперевода определённый исходный язык запоминается на время сессии. Повторная генерация с тем же промптом/источником/целевым языком использует кэш клиента, не обращаясь к сервису перевода повторно.

## Ограничения

- Эта версия пока не защищает сложные токены промптов (LoRA-теги, группы весов) плейсхолдерами.
